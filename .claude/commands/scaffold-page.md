Create `src/app/dashboard/$ARGUMENTS/page.tsx` as a server component.
Brand rules from `docs/BRAND.md`:

- Background `#0E0E10` (page), `#15151A` (surface), `#1F1F25` (raised).
- Brand accent gold `#E5B85A` for primary CTAs / highlights.
- Status: success → mint `#16D060`, danger → coral `#E73E54`.
- Stage colors only on stage chips: gold (Main), blue (Alt), violet
  (Pool), pink (Collectives).
- Border radius `rounded-md` only. Borders not shadows.
- Geist Sans for everything; Geist Mono for numbers/dates/IDs;
  Newsreader DISPLAY ONLY (wordmark + page H1).
- No emoji. No exclamation points.
